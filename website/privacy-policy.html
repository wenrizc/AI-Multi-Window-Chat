Privacy Policy

Last updated: January 14, 2026

AI Multi-Window Chat is an open-source browser extension. This privacy policy explains how we handle your data.

Data Collection

The extension collects the following data, stored locally in your browser:

  • API configuration (API keys, endpoints, model names)
  • Chat history
  • Custom prompts
  • Settings and preferences

Data Usage

Your data is used only for:

  • Communicating with your configured API service
  • Local storage and management of chat history
  • Saving your configuration settings
  • Exporting conversation records

We do not use your data for advertising, analytics, or sell to third parties.

Data Storage

All data is stored locally on your device using Chrome storage API. The extension does not use any cloud servers or databases.

When you send messages, data is sent directly to your configured API endpoint (e.g., OpenAI), and responses are stored locally.

Data Sharing

We do not share your data with third parties. When using third-party AI services, your data is sent directly to that service and is subject to their privacy policies.

Please review the privacy policies of any AI services you use.

Data Deletion

You can delete data through:

  • Deleting individual conversations in the extension
  • Clearing all history in the extension
  • Uninstalling the extension to remove all data

Data Security

We implement reasonable security measures to protect your data:

  • Data stored in browser, protected by browser security
  • Data sent only to configured API endpoints
  • Open source code available for security audit

Please note: anyone with access to your device and browser may view stored data.

Policy Changes

We may update this privacy policy. Changes will be posted on this page.

Contact Us

For questions or requests, please contact us via GitHub Issues.

隐私政策

最后更新:2026年1月14日

AI 多窗口聊天是一款开源浏览器扩展。本隐私政策说明我们如何处理您的数据。

数据收集

本扩展收集以下数据,全部存储在您的浏览器本地:

  • API 配置(API 密钥、端点、模型名称)
  • 聊天记录
  • 自定义提示词
  • 设置和偏好

数据使用

您的数据仅用于:

  • 与您配置的 API 服务通信
  • 本地存储和管理聊天记录
  • 保存您的配置设置
  • 导出对话记录

我们不会将您的数据用于广告、分析或出售给第三方。

数据存储

所有数据使用 Chrome storage API 本地存储在您的设备上。本扩展不使用任何云服务器或数据库。

当您发送消息时,数据直接发送到您配置的 API 端点(例如 OpenAI),响应存储在本地。

数据共享

我们不会向第三方共享您的数据。使用第三方 AI 服务时,您的数据直接发送到该服务,受其隐私政策约束。

请审查您使用的任何 AI 服务的隐私政策。

删除数据

您可以通过以下方式删除数据:

  • 在扩展中删除单个对话
  • 在扩展中清空所有历史记录
  • 卸载扩展以删除所有数据

数据安全

我们采取合理的安全措施保护您的数据:

  • 数据存储在浏览器中,受浏览器安全机制保护
  • 数据仅发送到您配置的 API 端点
  • 代码开源,可接受安全审计

请注意:任何有权访问您设备和浏览器的人都可以查看存储的数据。

政策变更

我们可能会更新本隐私政策,变更将在此页面发布。

联系我们

如有疑问或请求,请通过 GitHub Issues 联系我们。
